## Property Listing Operation

1. A logged-in Pro user submits a listing from `/submit-property`.
2. The listing is saved as `PENDING`.
3. Admin opens `/admin/listings`.
4. Admin reviews the content.
5. Admin clicks:
   - `公開する` -> `ACTIVE`
   - `却下する` -> `REJECTED`
   - `非公開` -> `EXPIRED`
6. Public APIs only expose `ACTIVE` listings.
